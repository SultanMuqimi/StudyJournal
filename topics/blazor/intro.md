# Blazor Intro

Blazor is a .NET web framework for building interactive web UIs with C# instead of JavaScript.

Key points:
- Runs in the browser via WebAssembly (`Blazor WebAssembly`) or on the server with SignalR updates (`Blazor Server`).
- Uses Razor components (`.razor` files) to combine C# and HTML in reusable UI blocks.
- Supports dependency injection, routing, data binding, and component events.

Why use Blazor?
- Share code between client and server using .NET.
- Strong typing and tooling with VS Code / Visual Studio.
- Fast development with live reload and component-based architecture.

This folder is a place to collect Blazor learning notes and small code samples.
